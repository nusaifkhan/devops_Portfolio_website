# Nusaif Khan — DevOps & Cloud Automation Portfolio

A dynamic, immersive, and fully front-end personal portfolio built to showcase my expertise and track record in Cloud Automation, Container Orchestration, Jenkins CI/CD Pipelines, and Site Reliability Engineering.

## 🚀 Overview

This portfolio leverages HTML, CSS, and lightweight client-side React (via CDN) to create a seamless user experience reminiscent of an IDE environment. With its clean layout, glass-morphic elements, and customized cursor functionality, the application feels fast, minimal, and fully interactive, highlighting projects, skills, certifications, and Fiverr freelancing services. 

100% of this branch is front-end functionality. The repository relies entirely on static assets, making it exceptionally fast to host!

## 📁 Repository Structure

```text
├── index.html              # Main homepage & routing logic (React)
├── style.css               # Shared global styling and utility classes
├── about.html              # Dedicated page detailing System Profile / Education
├── skills.html             # Categorized skill tags (CI/CD, Monitoring, Tools)
├── projects.html           # Deep dive into engineering / coding projects
├── cover-letter.html       # Digital customized cover letter experience
├── certifications.html     # Cloud & container engineering certs catalog
├── fiverr.html             # Fiverr freelance gig highlights & statistics
├── learning-path.html      # Track progress on upcoming goals & skills
└── Nusaif_Khan_Resume.pdf  # Downloadable static resume asset
```

## ⚡ Deployment on Vercel

If you would like to showcase this project to the world, it is primed for zero-configuration deployments using **Vercel**, **Netlify**, or **GitHub Pages**. Since it is entirely static and stateless, getting it online takes under 2 minutes:

### Vercel Quick Start

1. **Push** this repository to your GitHub account.
2. Log in directly to [Vercel.com](https://vercel.com/) via your GitHub credentials.
3. Click "Add New" -> "Project" from your Vercel Dashboard.
4. **Import** the GitHub repository containing these files.
5. In the Build and Output Settings, leave the default fields empty as Vercel natively detects and serves static `.html` frontends.
6. Hit **Deploy**. 

Vercel will immediately map the repository to a fast, global CDN edge network edge domain securely behind HTTPS.

## 💻 Local Development

If you desire to run or change this locally quickly without pushing to real servers immediately, simply run:
```bash
npx serve .
```
And view the application instantly on your `localhost:3000`.
